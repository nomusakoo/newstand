⚡ GitHub Issues 연동 방법

  • 01 GitHub에 새 repo 생성 (예: my-insights) — Public 권장
  • 02 repo → Labels 탭에서 라벨 생성: investment, blog, watch, idea
  • 03 Issues 탭 → 새 Issue 작성 → 라벨 investment 부착
  • 04 위 입력창에 GitHub username과 repo명 입력 → "저장 & 로드"
  • 05 이 파일을 index.html로 GitHub Pages 배포 (Settings → Pages → main/root)

✍️ 블로그 포스팅 인사이트 활용법

  • 01 투자 인사이트와 동일 repo 사용 가능 (라벨만 다르게)
  • 02 라벨 blog: 포스팅 초안 & 아이디어 메모

  • 04 Issue 닫기(Close) = 포스팅 완료 처리

⚙️ RSS Worker 설정

Cloudflare Worker URL을 등록하면 RSS를 횟수 제한 없이 직접 가져올 수 있어요.

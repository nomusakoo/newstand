⚡ GitHub Issues 연동 방법

  • 01 GitHub에 새 repo 생성 (예: my-insights) — Public 권장
  • 02 repo → Labels 탭에서 라벨 생성: investment, blog, watch, idea
  • 03 Issues 탭에서 새 Issue 작성 → 제목·내용 작성 → 라벨 investment 부착
  • 04 위 입력창에 GitHub username과 repo명 입력 → "저장 & 로드" 클릭
  • 05 이 HTML을 index.html로 저장해 GitHub Pages 배포 (Settings → Pages → main/root)

✍️ 블로그 포스팅 인사이트 활용법

  • 01 투자 인사이트와 같은 repo 사용 가능 (라벨만 다르게)
  • 02 라벨 blog: 포스팅 아이디어 / 초안 메모
  • 03 라벨 idea: 키워드 리서치 / 트렌드 메모
  • 04 Issue 본문에 마크다운으로 구성 → GitHub에서 바로 편집 가능
  • 05 Issue 닫기(Close) = 포스팅 완료 처리
